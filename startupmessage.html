Jamba Release Notes / News


Cross Realm Support Added


* Jamba 6.0.3-1 and up have cross realm character support.

* Character names now must be in name-server format, .e.g. Jafula-Caelestrasz

* Cross realm teams must make sure /jamba > Advanced > Core: Communications > Assume All Team Members Always Online is checked. Remember to reload your UI after making this change.

* This introduced quite a few bugs. Sorry :-( I'm working on fixing them as I am made aware of them.


Having Problems With Your Teams?


* Remember to do the following fixes on every character in your team!

* If your team list is full of empty lines, please reset the current profile. Warning this will require you to configure everything again. You can reset the current profile with the following button: /jamba > Profiles > Core: Profiles > Reset Profile

* Make sure to add characters into your team list with their name, followed by a dash and the server the character is on. e.g. Jafula-Caelestrasz


ISBoxer Users


* Please make sure your ISBoxer version is ISBoxer 41.10.1115.1 or higher.

* When ISBoxer manages your Jamba Team for you, it will add the characters with the correct name format if you have filled out the server name in your ISBoxer character.

Change Log


6.0.3-3


Core
* Various fixes for party leader changing and loot options.
* Push team settings now push the options "Override: Set loot to Group Loot if stranger in group." and "Friends Are Not Strangers"
* Override: Set loot to Group Loot if stranger in group no longer considers team members to be strangers.

Jamba-Display-Team
* Character level should display properly again now.
* Clique should still work with the health and power bars now.
* Health and power bars should update correctly.

Jamba-Quest
* Accept shared quest from team should work again.

Jamba-Toon
* If master not focus / target warnings should work again.

Known Issues
* The functions on the quest log bar (abandon all, share all, etc) may no longer work properly.


6.0.3-2


Core
* Same realm teams can again now use the channel for team online/offline checks. Cross realm teams must use the assume characters always online option.
* When adding a character to the team, will automatically append realm name if one does not exist.
* When master changes, party leader will now change as well if this option is selected.
* When focus changes, master will update to focus if this option is selected.

Jamba-Display-Team
* Clique should work with the health and power bars now.

Jamba-Follow
* Follow strobing now works with new name-server format.

Jamba-Trade
* Trade now works with new name-server format.

Known Issues
* The functions on the quest log bar (abandon all, share all, etc) may no longer work properly.


6.0.3-1


Core
* Initial cross realm support.
* Slaves renamed to minions.
* Attempt to increase character to character addon message performance by only sending messages once (sometimes messages were being sent twice).

Jamba-Quest-Watcher
* Quests now remove themselves from the quest watcher when they have been completed.
* Popup quests are back, but without the fancy frame.

Item-Bar
* Forked LibActionButton library in attempt to help compatibility with ElvUI.

Known Issues
* The functions on the quest log bar (abandon all, share all, etc) may no longer work properly.
